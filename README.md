# Todoey-Core-Data
A to-do list iOS app utilizing Core Data

* UITableView and Delegate
* Core Data (CRUD)
* UISearchBar and Querying with Core Data
* Swipeable table cells with [SwipeCellKit][swipecellkit-url]
